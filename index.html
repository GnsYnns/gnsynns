02

Stats Intelligence

Business Intelligence • LLM • Data Analysis 🟢 En développement actif

Analyse statistique avancée de données de livraison : 6 variables (distance Haversine, durée, poids, volume, temporalité, satisfaction). Tests d'hypothèses (T de Student, Welch), intervalles de confiance 95%, corrélations. Architecture MVC avec génération automatique de rapports PDF LaTeX professionnels.

Python Pandas NumPy SciPy Matplotlib LaTeX Streamlit (futur) LLM (futur)

Cas d'usage : impact Black Friday sur livraisons, corrélation poids/satisfaction. En développement : module LLM pour interprétation automatique + dashboard Streamlit interactif. Transformation d'un projet académique en outil professionnel de Data Analytics.

03

Nantes City Bot

RAG • Open Data • Agent IA 🔵 Roadmap 2026

Agent conversationnel intelligent connecté aux APIs Open Data de Nantes. Architecture RAG pour répondre en temps réel aux questions sur les transports (TAN), parkings P+R, événements culturels, et services municipaux. Rendre les données publiques accessibles via interface de chat en langage naturel.

LangChain FastAPI ChromaDB Nantes Open Data API RAG

Projet planifié pour 2026. Objectif : démontrer la maîtrise complète des agents IA autonomes, vectorisation de données, et consommation d'APIs publiques complexes. IA citoyenne parlant français naturellement.
